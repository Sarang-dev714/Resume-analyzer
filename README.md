# Resume Analyzer
AI-based Resume Analyzer built using Python and Flask to evaluate resumes based on ATS (Applicant Tracking System) logic.

---

##  Features

- Resume parsing (PDF support)
- Smart skill detection using keyword matching
- ATS Score calculation
- Missing skill identification
- Resume improvement suggestions
- Resume section breakdown (Education, Experience, Skills, etc.)

---

##  Tech Stack

- Python (Flask)
- HTML, CSS
- PyPDF2
- Regex (for smart matching)

---

##  How to Run

```bash
pip install flask PyPDF2
python app.py
